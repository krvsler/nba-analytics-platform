# NBA Analytics Platform

A full-stack web application for tracking NBA players and team performances through interactive, database-driven dashboards.

## Live Demo
_Coming soon!_

## Features 
- Player stat explorer with search and filtering features
- Team performance dashboard
- H2H player comparisons
- Charts operating season trends
- Top performers leaderboard
- CSV data export

## Tech Stack 
- **Frontend:** React, Vite, Recharts
- **Backend:** Node.js, Express
- **Database:** PostgreSQL (Supabase)
- **Auth:** JWT
- **Deployment:** Vercel (frontend), Render (backend)

## Data Source
NBA stats via BallDontLie API
